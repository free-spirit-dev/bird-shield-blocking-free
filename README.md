## 《Twitter鸟盾》封锁名单共享仓库

此仓库用于自由发布名单,<br/>
愿意发布名单的小伙伴请与我的Twitter私信

### 关于《Twitter鸟盾》

那些黄推本质为“鸟事”，  
Twitter为“蓝鸟”，  
且马斯克这“鸟人”不管这“鸟事”事  
所以定为《Twitter鸟盾》

### 目前已定会开发功能：
1. 可以和朋友共享自定义屏蔽库  
2. 可以选择是隐藏还是加入黑名单2种方式屏蔽  
3. 可以根据用户名和内容关键字2种方式屏蔽
